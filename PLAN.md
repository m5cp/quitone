# QuitOne — Calm Habit Progress Tracker

## Features

- **One-tap daily check-in** — open the app, see your progress, confirm your day with a single tap
- **Onboarding in under 60 seconds** — pick a habit, set daily spend, choose a goal, and start
- **Current run & total progress tracking** — always know how many days you've been on track
- **Money saved calculator** — see estimated spending avoided based on your daily input
- **Slip recovery** — a supportive "I had a slip" button that resets your current run but keeps total progress, with kind messaging
- **Daily rotating insights** — gentle motivational messages that rotate each day
- **Progress screen** — visual breakdown of current run, total progress, and money saved over time
- **Profile & settings** — edit habit, daily spend, toggle notifications, and see your overall journey
- **Daily notification** — one calm reminder per day (user can disable)
- **Fully private** — all data stored on-device only, no account needed

## Design

- **Light & minimal** — clean white background with soft warm touches, lots of whitespace
- **System blue accent** — classic iOS feel for buttons and interactive elements
- **Large, readable typography** — SF Pro with big numbers for key metrics, clear visual hierarchy
- **Rounded cards** with subtle shadows for the hero stat display
- **Generous spacing** — 16–24pt padding, no cramped layouts
- **Subtle spring animations** on check-in and progress updates
- **Haptic feedback** on daily check-in taps
- **Inspired by Apple Health, Calm, and Streaks** — premium, native, effortless

## Screens

1. **Onboarding flow** (full-screen, 4 steps)
  - Step 1: Pick a habit from a grid of options (Smoking, Vaping, Alcohol, etc.) or type custom
  - Step 2: Set daily spend with quick-select chips ($5, $10, $15, $20, Custom)
  - Step 3: Choose goal — "Stop completely" or "Reduce over time"
  - Step 4: "You're ready to start." with a "Begin" button
2. **Home screen** (main tab)
  - Habit name and supportive status message at top
  - Large hero card showing current run days (big number), total progress, and estimated money saved
  - Primary button: "I stayed on track today"
  - Secondary button: "I had a slip — keep going"
  - Rotating daily insight at bottom
3. **Progress screen** (second tab)
  - Current run and total progress stats
  - Estimated money avoided with a simple visual chart
  - Units avoided count
  - Empty state: "Your progress will build here."
4. **Profile screen** (third tab)
  - Header with supportive message
  - Habit settings (type, daily spend)
  - Notification toggle
  - Reset option

## App Icon
